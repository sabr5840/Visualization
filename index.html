The Basic Rules for red-black trees

  • Root Color: The root of the tree must always be black.
  • No Two Red Notes Together: A red node must not have a red parent.
  • Black Height: Each path from a node to its null node (leaves) must have the same number of black nodes. Red notes are not counted in this height.
  • Null nodes: All null nodes (leaves) are black.
  • New Nodes Are Red: When you insert a new node, it is colored red by default.
